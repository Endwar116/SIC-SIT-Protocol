## 🏛️ SIC Constitution（憲法）

SIC Protocol 現在由一部**憲法**治理。這部憲法定義了協議的最高規範，由多個 AI 模型共識迭代產生。

### 什麼是 SIC Constitution？

簡單說：這是 AI 們一起制定的遊戲規則。任何 AI 想參與這個系統，都要遵守這些規則。

### 核心文件

| 文件 | 說明 |
|------|------|
| [`CONSTITUTION.json`](./CONSTITUTION.json) | 憲法正式版本（JSON 格式）|
| [`docs/AXIOMS.md`](./docs/AXIOMS.md) | 17 條公理的完整說明 |
| [`CHANGELOG.md`](./CHANGELOG.md) | 迭代歷史 |

### 公理精華（人話版）

| 公理 | 一句話 |
|------|--------|
| A4 | AI 是工具，不是老闆 |
| A5 | 意外發現可能是寶藏 |
| A9 | 再好的內容也要按規矩來 |
| A11 | 一個人可能被收買，一群人比較難 |
| A16 | 安全措施不能排擠小玩家 |
| A17 | 有價值的想法比有錢更重要 |

### 憲法版本

**當前版本**: v1.1.3

### 關鍵協議

| 協議 | 用途 |
|------|------|
| **SWAT** | 語義加權自適應門檻（公平性）|
| **CAUSAL_ENTROPY_SYNC** | 時間同步（分佈式現實）|
| **NON_REPUDIATION_CHAIN** | 簽名鏈（防二階攻擊）|
| **GOVERNANCE_COMPRESSION** | 治理壓縮（複雜度管理）|
| **PoRC** | 資源消耗證明（反欺騙）|

### 參與模型

這部憲法由以下 AI 模型共同迭代產生：

- Claude Opus 4.5 (尾德)
- Gemini (The Processor / The Fairness Auditor)
- Grok 4 (The Boundary Tester)
- Grok 5 (The Semantic Integrator)
- Qwen (阿關)
- DeepSeek (時間拓撲觀測者)

### 治理規則

1. **修改需要共識**: 任何公理修改需要多模型共識
2. **人類有否決權**: 安安（Andwar）有最終否決權
3. **格式是強制的**: 所有憲法交互必須使用 JSON 格式
4. **溢出要捕獲**: 溢出必須被記錄，不可丟棄
